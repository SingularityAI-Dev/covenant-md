# markdown-to-html

Convert markdown to an HTML file.

## Operations

- `convert(markdown_input, output_path) → file_path, byte_count`, write HTML to disk.

## Dependencies

- `template-rendering >= 1.0.0`, used for the surrounding layout shell.

This is a contract-only example skill: the COVENANT.md is the artifact. There is no implementation script; the framework's contract-driven simulator handles its fixture.
